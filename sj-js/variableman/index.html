1. Getting Started

1-1. How to load?

  • Browser
      <script src="https://cdn.jsdelivr.net/npm/@sj-js/crossman/dist/js/crossman.min.js"></script>
  <script src="https://cdn.jsdelivr.net/npm/@sj-js/variableman/dist/js/variableman.min.js"></script>
  <script>
       var variableman = new VariableMan();
  </script>
  • ES6+
      npm install @sj-js/variableman

      const VariableMan = require('@sj-js/variableman');
  const variableman = new VariableMan();
